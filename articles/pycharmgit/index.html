Connect Your PyCharm to Github and Auto Push & Pull
PyCharm链接GitHub并自动同步

1. Download Git Bash and install.

If Git Bash appears when you right click on desktop, then it's successfully installed

2. Download PyCharm and install.

3. Open PyCharm -> File -> Settings -> Version Control -> Git, select your Git Bash path.

In my PC, it's C:\Program Files\Git\cmd\git.exe

4. In PyCharm -> VCS -> Git, Select your project’s repository then create a local folder.

5. If you want to push, VCS -> Git -> push, then commit and push.
