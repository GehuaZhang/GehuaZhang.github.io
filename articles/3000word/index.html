GRE 3000 Words App for MacOS

2017-06-14
Programming

This is an application for helping you remember GRE words. The words are coming from book “再要你命3000(第二版)”. This application supports only on MacOS for now.

How to use

Download zip file from here: GRE Words 3000 (国内可能需要翻墙)

unZip the file and click 3000.app

Choose the list you want to study, select mode to “Review” or “Learn”. (You need to learn a full list first then you will be allowed to review)

Start to learn words.

If you know this word, click “Know”, if not, click “Don’t know”.

Then it appears the Chinese meaning of this word.

If it matches your memory, click “Correct”, if not, click “Not Correct”.

After you finish this list, words you don’t know appear in a list.

Behind App

I used Python 2.7 to build this project, with original pack “tkinter”. Full code is here Python File
PyInstaller helped me package all files and GUI.

Problems encountered:

  1. PyInstaller does not support Python 3.6+, be careful with the interpreter.

  2. Terminal/Shell in MacOS should be converted to Python 2 first.

  3. Do not directly generate app file. Generate a SPEC file first then configure it with a new directory of the Excel file. Next use SPEC to generate app.

  4. IMPORTANT If your PyInstaller says ImportError: dlopen(.../dist/script/_tkinter.so, 2): Library not loaded: @loader_path/Tcl, follow this:

    • Change two files in PyInstaller, by this instruction: Instruction

    • Copy and paste these two files to your dist: /System/Library/Frameworks/Tk.framework/Versions/8.5/Tk and /System/Library/Frameworks/Tcl.framework/Versions/8.5/Tcl
